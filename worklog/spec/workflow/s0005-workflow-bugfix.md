# Workflow: Bug Fix

Correcting defective behavior. Flow: task → spec (if gap revealed).

## Requirements

- A regression test that fails before the fix and passes after.
- Root cause is identified, not just the observed symptom.
- If the bug reveals a spec gap, the spec is updated.
